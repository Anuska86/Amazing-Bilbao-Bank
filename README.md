🏦 Amazing Bilbao Bank:

Welcome to my first major Java project! This application is a full-stack banking system that I built while learning the fundamentals of the language and moving into web development.

🎓 The Learning Journey:

This project represents my progression through the Java ecosystem:

- Phase 1: Vanilla Java: I started by building the core logic using standard classes, OOP principles (Inheritance, Polymorphism), and Collections.

- Phase 2: Database Connectivity: I integrated MySQL using JDBC, learning how to manage connections securely with try-with-resources.

- Phase 3: Web Integration: I moved into the Jakarta EE world, using Servlets for the controller logic and JSP/JSTL for the dynamic frontend.

🚀 Key Features: 

- MVC Architecture: A clean separation between Models (Accounts/Transactions), the Controller (MainController), and the View (JSP files).

- Transaction Safety: Implementation of SQL Transactions with commit() and rollback() to ensure data integrity during money transfers.

- Session Management: Secure user login and logout functionality.

- Dynamic Dashboard: Real-time balance updates and transaction history categorization (Internal vs. External).

🛠️ Tech Stack:


- Language: Java 26 (The latest version!)

- Backend: Jakarta Servlets, JSP, JSTL

- Frontend: Bootstrap 5 (For responsive, mobile-friendly design), CSS3

- Database: MySQL 9.0

- Server: Apache Tomcat 10.1

- Build/IDE: Eclipse IDE

🔧 Installation & Setup:


Clone the Repository:

Bash

git clone [https://github.com/Anuska86/MyFirstProject]

Database Configuration:

- Import the provided SQL script to create the amazing_bilbao_bank database.

- Set an environment variable named DB_PASSWORD with your MySQL password.

Driver Setup:

- Ensure mysql-connector-j-9.6.0.jar is present in src/main/webapp/WEB-INF/lib.

Run:

- Deploy the project onto an Apache Tomcat 10.1 server.
